# 🌍 Climate Trend Analysis Dashboard

## 📌 Project Overview

This project analyzes climate trends using historical environmental data to identify patterns in temperature, CO₂ concentration, rainfall, humidity, and temperature anomalies across different regions. The goal is to transform raw climate data into actionable insights through SQL analysis, Python-based exploratory data analysis (EDA), and an interactive Power BI dashboard.

---

## 🎯 Business Problem

Climate change is one of the biggest global challenges. Governments, researchers, and organizations need data-driven insights to monitor environmental changes and support sustainable decision-making.

This project answers questions such as:

- Which regions have the highest average temperatures?
- How has CO₂ concentration changed over time?
- Which regions receive the highest rainfall?
- How does humidity vary across regions?
- Are temperature anomalies increasing over time?

---

## 📂 Dataset

The dataset contains **5,000 climate observations** with the following columns:

| Column | Description |
|---------|-------------|
| Date | Observation date |
| Region | Geographic region |
| Average Temperature (°C) | Average daily temperature |
| Precipitation (mm) | Rainfall |
| CO₂ (ppm) | Atmospheric CO₂ concentration |
| Temperature Anomaly (°C) | Difference from baseline temperature |
| Humidity (%) | Relative humidity |
